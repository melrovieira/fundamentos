Wifi Default Pass Wordlist Generator


Este script gera uma lista de palavras com base nas senhas padrão de Wi-Fi usadas por alguns ISPs (Provedores de Serviços de Internet),
como Vivo e Claro, no Brasil. Esses provedores costumam usar senhas padrão derivadas do endereço MAC ou do nome da rede.
A ferramenta cria combinações que podem ser úteis para testes de segurança de rede (por exemplo, quebra de senhas, auditoria)
em redes onde as senhas padrão ainda são usadas.

Wasteland-Scavenger
